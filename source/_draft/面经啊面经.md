#### 使用 const 声明一个对象，对象可改吗，为什么

在 JavaScript 中，使用 const 声明的变量是常量，这意味着其引用的内存地址不可变，但对象本身的属性是可以修改的。这意味着，虽然你不能重新分配一个 const 变量的值，但你可以修改这个变量引用的对象的属性。
